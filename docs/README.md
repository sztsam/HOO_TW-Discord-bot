HOO is a Tribal Wars bot that enchances gameplay using discord.


Features:
  - Multi-region and multi-world support
  - Multi language support
  - Live tribe conquer alerts
  - Village info
  - Player info
  - Tribe info
  - Circular lists
  - Graph generation
  - World map generation
  - AI chat
  - Punish users / swear words
